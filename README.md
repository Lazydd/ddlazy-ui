## Project setup
```
npm install ddLazy-ui
```

### Quick Start
```
import Vue from 'vue'
import ddlazyui from 'ddlazy-ui'

Vue.use(ddlazyui)

// or
import {
  Select,
  Button
  // ...
} from 'ddlazy-ui'

Vue.component(Select.name, Select)
Vue.component(Button.name, Button)
```

## Browser Support
Modern browsers and Internet Explorer 10+.
